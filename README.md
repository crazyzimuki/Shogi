# Shogi Project
**Author:** Ofir Cohen

---

## Description
This project contains a MiniShogi Android app and a MiniShogi/Full Shogi desktop program.  
Features include:
- Play against AI with adjustable difficulty levels  
- Undo move functionality  
- Available on Android (.apk) and Windows (.exe)  

---

## How to Run

### Android (MiniShogi)
1. Install Mini-Shinobi.apk on your device.  
2. Launch the app and play against the AI. 

### Windows (MiniShogi & Full Shogi)
1. Download the Release folder.
2. Run Mini-Shinobi-Beta.exe to play MiniShogi or open the folder "Shinobi-Full" and run the file Shinobi.exe to play full-size Shogi!

---

## Features
- AI opponent with multiple difficulty levels  
- Undo moves at any time  
- Intuitive interface for both Android and PC
